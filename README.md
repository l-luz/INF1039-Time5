# INF1039---Time5

Descrição curta do projeto.

## Pré-requisitos
* Node.js
* npm
## Instalação

1. Clone o repositório:
   ```
   git clone https://github.com/l-luz/INF1039-Time5.git
   ```

3. Navegue até o diretório do projeto:

   ```
   cd time5-proj
   ```

5. Instale as dependências:

   ```
   npm install
   ```

## Uso

   ```
  npm run dev
  ```
